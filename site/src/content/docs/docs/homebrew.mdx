---

macOS / Linux では Homebrew で `tdsl` を導入できます。

```sh
brew tap keroway/tap
brew install tdsl
tdsl --help
```

更新する場合は通常の Homebrew コマンドを使います。

```sh
brew update
brew upgrade tdsl
```

## ワンラインインストール

Homebrewを使わない場合は、リリース済みバイナリを取得するインストールスクリプトを使えます。

```sh
curl -sSfL https://raw.githubusercontent.com/keroway/timeline-dsl/main/install.sh | sh
```

インストール先は既定で `~/.local/bin/tdsl` です。`~/.local/bin` が `PATH` に入っていない場合は、利用しているシェルの設定ファイルに追加します。

```sh
export PATH="${HOME}/.local/bin:${PATH}"
```

## Windows（バイナリ手動インストール）

Windows では Homebrew が使えないため、GitHub リリースページからバイナリを手動で取得します。

1. [GitHub Releases](https://github.com/keroway/timeline-dsl/releases/latest) を開きます。
2. Assets から `tdsl-windows-x86_64.zip` をダウンロードします。
3. ZIP を展開し、`tdsl.exe` を任意のフォルダ（例: `C:\Users\<ユーザー名>\bin`）に配置します。
4. そのフォルダを `PATH` に追加します。

   **PowerShell（ユーザースコープ）:**

   ```powershell
   $dir = "$env:USERPROFILE\bin"
   [System.Environment]::SetEnvironmentVariable("Path", "$([System.Environment]::GetEnvironmentVariable('Path','User'));$dir", "User")
   ```

5. 新しいターミナルを開いて動作確認します。

   ```powershell
   tdsl --help
   ```

## cargoでインストール

Rust開発環境がある場合は、GitHubリポジトリから `tdsl-cli` をインストールできます。

```sh
cargo install --git https://github.com/keroway/timeline-dsl tdsl-cli
```
