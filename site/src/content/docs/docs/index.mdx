Timeline DSL は、年表をテキストで記述し、PlaygroundやCLIで検証・生成するためのツールです。

このドキュメントでは、Timeline DSL 利用者に向けて、文法サンプル、最初の `.tdsl` ファイル作成、Playgroundでの確認、CLIでのビルド、HTMLレンダリング、CIでのチェックまでを扱います。

## 想定する使い方

- 歴史やプロジェクトの出来事をGit管理する
- 年表データをレビュー可能なテキストとして扱う
- CIやリリースフローで年表の整合性を確認する
