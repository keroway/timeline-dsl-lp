Timeline DSL は、年表をテキストで記述し、CLIで検証・生成するためのツールです。

このドキュメントでは、インストール、最初の `.tdsl` ファイル作成、CLIでのビルド確認、HTMLレンダリング、CIでのチェックまでを扱います。

## 想定する使い方

- 歴史やプロジェクトの出来事をGit管理する
- 年表データをレビュー可能なテキストとして扱う
- CIやリリースフローで年表の整合性を確認する
