---
title: インストール
description: Timeline DSL CLI の導入方法です。
---

Timeline DSL はCLIツールとして利用します。初期サイトでは Homebrew 経由の導入を前提に案内します。

```sh
brew tap keroway/tap
brew install tdsl
```

インストール後、CLIが実行できることを確認します。

```sh
tdsl --help
```

## 必要な環境

- macOS または Linux
- Homebrew
- Git

Homebrew 以外の導入方法は [Homebrewとその他の導入方法](/docs/homebrew/) を参照してください。
