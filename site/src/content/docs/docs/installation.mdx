---
title: インストール
description: Timeline DSL CLI の導入方法です。
---

Timeline DSL はCLIツールとして利用します。初期サイトでは Homebrew 経由の導入を前提に案内します。

```sh
brew install keroway/tap/timeline-dsl
```

インストール後、CLIが実行できることを確認します。

```sh
tdsl --help
```

## 必要な環境

- macOS または Linux
- Homebrew
- Git

他の配布経路は、ツール本体のリリース方針に合わせて後続で追加します。
