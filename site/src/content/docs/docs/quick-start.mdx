---
title: Quick Start
description: Timeline DSL で最初の年表を作る手順です。
---

最小の `.tdsl` ファイルを作り、CLIで検証します。

```txt
timeline "Sample"

event "Kickoff" {
  date: 2026-01-01
  lane: "Project"
}
```

保存したファイルをCLIでビルドします。

```sh
tdsl build sample.tdsl --pretty
```

出力が成功すれば、年表データとして解釈できています。

## 次のステップ

- 複数イベントを追加する
- レーンで分類する
- CIで `tdsl build` を実行する
